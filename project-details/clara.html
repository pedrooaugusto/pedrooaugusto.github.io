Implementação de uma sala de chat com versões para Windows e Android. Funciona de maneira P2P, ou seja, não depende de conexão com a internet, basta que os dois ou mais aparelhos estejam conectados a mesma rede para funcionar. Um dos aparelhos faz o papel de cliente/servidor e os outros podem se conectar a ele como clientes.

Tecnologias

  • Java
  • Swing
  • Sockets
  • Android
  • Threads
