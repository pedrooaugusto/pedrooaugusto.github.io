Github DevShop

Descrição

  • Pedro Augusto

  • https://githubdevshop.herokuapp.com

  • https://github.com/pedrooaugusto/devshop

  • 20 dezembro, 2017

Implementação de um carrinho de compras usando MERN Stack, onde os items são os desenvolvedores de uma organização qualquer do Github. O preço de cada desenvolvedor é calculado com base em sua popularidade no Github (número de estrelas, número de repositórios e número de seguidores).

Tecnologias

  • NodeJS
  • Express
  • MongoDB
  • React
  • Material UI
