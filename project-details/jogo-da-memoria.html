Jogo da Memória

Descrição

  • Pedro Augusto

  • source on Github

  • 01 fevereiro, 2016

Implementação do jogo da memória no qual pode-se jogar com qualquer imagem. Ao iniciar, o programa procura por uma determinada pasta no diretório em que se encontra, dentro desta pasta devem conter as imagens disponíveis na hora do jogo. Dessa forma ao colocar novas imagens dentro da pasta elas automaticamente estarão disponíveis para jogar.

Tecnologias

  • Java
  • Swing
  • Images
  • Threads
