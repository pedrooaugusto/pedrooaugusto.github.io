Atari Breakout

Descrição

  • Pedro Augusto

  • live demo

  • source on Github

  • 10 janeiro, 2016

Implementação do clássico jogo Breakout da Atari. Uma combinação de JavaScript, HTML e bastante matemática. Existem 3 versões desse jogo: a versão Web que roda no navegador, foi feita usando HTML, JS e Canvas, a versão Android, feita com Java e a versão desktop, também feita com Java.
